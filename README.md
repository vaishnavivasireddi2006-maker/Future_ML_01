# 📊 Sales Forecasting using Machine Learning

## 📌 Project Description
This project predicts future sales using historical sales data.  
Machine Learning is used to analyze past trends and forecast future sales.

## 🎯 Objective
- Analyze past sales data
- Identify trends
- Predict future sales
- Visualize the results

## 🛠 Tools Used
- Python
- Jupyter Notebook / Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🔄 Project Steps
1. Data Cleaning
2. Feature Engineering
3. Data Visualization
4. Model Training (Linear Regression)
5. Sales Forecasting
6. Model Evaluation

## 📊 Results
The model forecasts sales for the next 6 months and visualizes the results using graphs.
